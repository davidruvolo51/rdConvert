# `format_rd_path`: Format Rd File name

## Description


 remove file extension from .Rd file names


## Usage

```r
format_rd_path(x)
```


## Arguments

Argument      |Description
------------- |----------------
```x```     |     a file name

## Examples

```r 
 format.rdpath(x = "man/test_file.Rd")
 
 ```
